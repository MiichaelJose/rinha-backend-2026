# Contexto do Projeto

Este projeto participa da Rinha de Backend 2026.

Regras:
- Priorizar performance e simplicidade
- Evitar abstrações desnecessárias
- Evitar ORMs pesados
- Preferir SQL direto
- Usar Docker Compose
- Otimizar uso de memória e CPU

Stack:
- Node.js
- Fastify
- PostgreSQL
- Redis
- Nginx

Objetivo inicial:
- Fazer funcionar
- Passar nos testes
- Depois otimizar

Diretrizes:
- Código simples
- Poucas dependências
- Rotas rápidas
- Evitar lógica síncrona pesada
- Evitar JSON gigante
- Reutilizar conexões
